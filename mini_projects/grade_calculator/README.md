# Grade Calculator

A Java mini project that calculates percentage and overall grade based on user-entered marks.

## Features
- Dynamic number of subjects
- Input validation
- Percentage calculation
- Grade calculation
- Exception handling
- Retry and exit system

## Concepts Used
- If-Else
- Loops
- Arrays
- Exception Handling
- User Input
- Scanner Class

## How to Run
Compile and run using:
```
javac GradeCalculator.java
java GradeCalculator
```

## Requirements
- Java JDK 8 or above

## Grade System
| Percentage | Grade |
|------------|-------|
|     90+    |   A+  |
|     80+    |   A   |
|     70+    |   B   |
|     60+    |   C   |
|     50+    |   D   |
|     40+    |   P   |
|  Below 40  |   F   |
